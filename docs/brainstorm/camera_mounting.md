# 相机安装方式与机械震动分析

## 1. 三种相机安装方式深度对比与风险评估

在面对滑台启停震动与标定漂移风险时，我们需要对三种可能的相机安装方式进行深度的多维度横向对比：
1. **安装方式 A：独立支架架设于地面（大地独立）**
2. **安装方式 B：固定于分拣机静止龙门架上（框架固定）**
3. **安装方式 C：臂载于平移滑台/机械臂末端（臂载移动）**

### 1.1 三大安装方案多维度横向对比表

| 对比维度 | 安装方式 A：独立支架立于大地 (Eye-to-Hand) | 安装方式 B：固定于分拣机框架 (Eye-to-Hand) | 安装方式 C：臂载于滑台机械臂 (Eye-in-Hand) |
| :--- | :--- | :--- | :--- |
| **Z轴测量精度** | **极差**<br>工作距离大（$Z \ge 800\text{ mm}$），误差高达 **$\pm 3.75\text{ mm}$**。对于 4mm 芦笋无法满足分层定位要求。 | **较差**<br>工作距离较大（$Z \ge 700\text{ mm}$），误差约 **$\pm 3.0\text{ mm}$**。卡缝动作精度不足。 | **优秀（推荐）**<br>工作距离小（$Z \le 400\text{ mm}$），误差 **$\le \pm 0.94\text{ mm}$**，拨钩接近芦笋时（$Z \approx 200\text{ mm}$, $\Delta d = 0.5\text{ px}$），误差约 **$\pm 0.24\text{ mm}$**。 |
| **物理视角遮挡** | **严重**<br>滑台平移到拨料中心区时，滑台自身的机械壳体和舵机拨钩会完全挡住相机对下方芦笋的视线。 | **严重**<br>与大地安装相同，滑台运动在拨钩下落卡缝的关键时刻，会对镜头产生完全的盲区遮挡。 | **无遮挡（推荐）**<br>相机与拨钩一体运动，拨钩和目标芦笋永远处于相机视野的中心，实现全流程实时监控。 |
| **机械振动影响** | **无振动（推荐）**<br>与机器完全物理隔离，不受分拣机自身运行震动的影响。 | **高频微震**<br>分拣机皮带和电机共振导致的高频微幅震动。可通过**全局快门 + 同步曝光（曝光 $\le 2\text{ ms}$）**完全消除。 | **低频大震**<br>滑台频繁加速、急停制动产生的低频大幅余震。必须通过程序在停机后**延时 50ms-80ms 消震**。 |
| **标定长期稳定性**| **高**<br>相机不运动且物理隔离，标定后几何状态稳定。 | **极高（推荐）**<br>相机与分拣机导轨基座刚性一体固定，几何状态极稳定。 | **有漂移风险**<br>滑台频繁加速、急停制动产生的惯性冲击，易导致相机悬臂支架产生微小变形。 |
| **滑台负载与节拍**| **无负载影响**<br>滑台重量极轻，电机响应快。可在运动中提前拍照定位，减少停止等待节拍。 | **无负载影响**<br>滑台重量极轻，电机响应快。可在运动中提前拍照定位，减少停止等待节拍。 | **增加负载**<br>增加了移动质量。滑台急停后必须等待物理余震收敛才能拍照，增加了分拣单次循环时间。 |
| **现场实施与空间**| **占地大**<br>需要在传送带旁独立立柱，额外占用现场生产空间。 | **空间优（推荐）**<br>一体化龙门架设计，无独立占地，随设备整体移动。 | **内部干涉风险**<br>臂载相机随滑台运动，运动轨迹上需要严防与高堆叠的芦笋或侧壁导轨产生内部物理刮擦。 |

### 1.2 固定框架方案的微震评估

针对固定于分拣机框架的方案（安装方式 B）所面临的自身震动传递，我们进行深入评估：

* **框架共振（安装方式 B）**：框架震动为高频微幅（$0.05\text{ mm}$ 级），可通过**全局快门（Global Shutter）**和**高强度外部光源**，将曝光时间压缩在 $2\text{ ms}$ 以内。即便框架有微震，也绝不会导致图像产生运动模糊或匹配失效。

> [!IMPORTANT]
> **本项目的物理红线决定了必须选用“臂载相机（安装方式 C）”**。
> 虽然固定安装标定更稳定，但是**严重的物理遮挡**和**测距精度平方律恶化（高度增大导致测距精度失效）**是无法克服的红线。臂载相机的“停机低频余震”和“标定漂移”可以通过**“CNC 铝合金硬质一体化相机座”**与**“硬同步曝光 + 50ms 延时消震”**以极低的成本在软件和结构上彻底防范。

## 2. 结构震动风险防范：频繁启停运动下的“双目标定漂移”

* **风险描述**：双目相机安装在运动滑台上。当步进电机带动滑台进行频繁的平移、制动时，虽然整体负载较小且无超高加速度，但滑台碰撞制动和急停瞬间的瞬间减速度依然会产生惯性力。若相机固定支架的刚性不足，长此以往会导致两个相机的相对夹角产生微小偏转。即使仅偏转 **0.05°**（相当于图像上 1-2 像素的错位），左右图像的极线对齐（Epipolar Rectification）就会失效，导致立体匹配深度图退化，产生 5-10mm 的测距误差。
* **防范措施**：
  1. 采用高刚性的 CNC 铝合金一体化切削支架，避免使用 3D 打印件或易变形的薄钣金弯折件。
  2. 优先选购一体式双目相机模块（两个感光芯片制备在同一块刚性 PCB 基板上），确保物理位姿在频繁震动下保持恒定。
  3. 软件中加入定期自校验机制：当滑台回到归零点（Homing Point）时，可拍摄固定标记物，自动校验并微调极线对齐矩阵。
